# N 찍기

<a href="https://www.acmicpc.net/problem/2741">백준 2741번 문제</a>

난이도 브론즈3


```java
import java.util.Scanner;

public class Main {

	public static void main(String[] args) {
		int n = new Scanner(System.in).nextInt();

		for (int i = 1; i <= n; ++i) {
			System.out.println(i);
		}
	}

}
```
